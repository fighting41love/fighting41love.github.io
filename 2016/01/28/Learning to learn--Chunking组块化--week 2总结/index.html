Learning to learn--Lesson 2 干货

2016-01-28 • 分类: 课程总结

Chunking 组块英文定义

Chunking: Chunking is about forming small mental patterns of meaning relating the material learned to a greater context. These chunks are committed to memory and are easy to retrieve when needed. They unite scattered information into retrievable conceptual units of meaning, so that we can think and perform actions smoothly without going into all the details each time.

组块概念

组块是大脑整理知识形成的信息片段。这些小的信息片段跟记忆相关，当你需要用到这个片段的时候，所以组块是大脑调动记忆的捷径 。
举个例子，“穿衣服”的过程是一个复杂的过程，拿起衣服，找好方向，找到袖子，再穿。“穿衣服”的过程，你不需要的大脑进行思考，因为你在大脑中形成了一个组块Chunk。当你穿衣时，自然的找到这个chunk并使用，不需要认真思考。也就是说，当你把自己的专业知识都组块化后，你运用专业知识就会非常自如了。

知识迁移

当然，组块之间是有联系的，比如 学习网球和乒乓球就有些类似，所以当你学习新的知识时，旧的组块可以帮助你在新的领域学习出新的chunk。这个被称为“Transfer”。跟机器学习中“Transfer Learning”的概念很像哈，所谓大道相通吧。

思维定势

另外，当形成了固定的组块后，当遇到新的问题时，人往往会用已有的组块解决新的问题，可能会造成思维定势（Einstellung）。所以，不仅要教大脑学会组块，也要学会自由切换组块。

组块对每个人来说非常重要，怎么教大脑形成组块呢?

  1. 集中注意力
  2. 理解知识
  3. 重复练习以帮助你掌握知识，构建知识整体框架
  4. 回忆（Recall）:脱离书本努力想起关键点！！！！！（这是促进组块化的最佳方式之一）
  5. 知识迁移（Transfer）
  6. Interleave 交替学习：选出各种不同的概念、方法和技术，同一时间段全部加以练习。

怎样证明你是真的“学会”了？还是在欺骗自己（Fouling yourself）。

  1. Test yourself and recall
  2. Minimize highlighting: 每个段落最多一句高亮标注核心idea，过多的highlighting句子，会影响你对核心idea的理解。更重要的是，当你在阅读高亮文本的时候，你会以为自己记住了那些尚未掌握的内容。
  3. Mistakes are good!
  4. Use deliberate practice. 不要只练习简单部分，这会给你一种自己已经完全掌握材料的错觉。有意练习你认为困难的部分。
